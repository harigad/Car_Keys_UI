ios_module_barcode_scanner Module

Barcode scanner

it works with: UPCECode, Code39Code, Code39Mod43Code, EAN13Code, EAN8Code, Code93Code, Code128Code,PDF417Code, QRCode, AztecCode

Accessing the ios_module_barcode_scanner Module

To access this module from JavaScript, you would do the following:

var ios_module_barcode_scanner = require("com.seanetix.BarcodeScanner");

The ios_module_barcode_scanner variable is a reference to the Module object.
